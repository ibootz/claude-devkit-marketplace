# 提案与讲解（Pitch & Explainer）

把本次工作（随调用传入的参数：要争取谁的支持 + 可用的产物）打包成一份提案与讲解文档。

## 服务两类读者

1. **带着和作者当初相同未知的评审者** —— 用讲解材料加速理解（背景、要解决的问题、方案概览）。
2. **专家** —— 展示已经考虑了他们会预料到的**未知点与常见故障点**，加速批准。

## 要求

- 把可用的**原型、规格说明、实现笔记**整合进**同一份文档**。
- **开头先放演示**（GIF/截图/可交互原型链接），让人一眼看到成果。
- 结构：演示 → 要解决的问题与背景 → 方案与关键决策 → 已考虑的未知/故障点/权衡 → 后续/风险。
- 产出一份能**直接丢进 Slack/群聊**的自包含文档（HTML 或 Markdown）。

## 原文示例 prompt（供理解意图）

- "把原型、规格说明和实现笔记打包成一份文档，让我能直接丢到 Slack 上争取支持。开头先放演示 GIF。"
